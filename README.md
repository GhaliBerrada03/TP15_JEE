# 📦 Spring Boot + Docker Project

## 🚀 Description

Ce projet est une application **Spring Boot** connectée à **MySQL**, déployée avec **Docker** et **Docker Compose**.

---

## ⚙️ Technologies utilisées

* Java 17
* Spring Boot
* Spring Data JPA
* MySQL
* Docker
* Docker Compose

---

## 🛠️ Prérequis

* Docker Desktop installé et démarré
* Java 17
* Maven (ou utiliser `mvnw`)
